### 3.1.1 Создайте список, в котором элементы могут быть выделены, как в файловых менеджерах. При клике на элемент списка выделяется только этот элемент (добавляется класс .selected), отменяется выделение остальных элементов. Если клик сделан вместе с Ctrl (Cmd для Mac), то выделение переключается на элементе, но остальные элементы при этом не изменяются. Демо: Кликни на элемент списка, чтобы выделить его.
