### 1.11.1 Найдите координаты точек относительно окна браузера В ифрейме ниже располагается документ с зелёным «полем». Используйте JavaScript, чтобы найти координаты углов, обозначенных стрелками. В документе уже реализована функциональность, когда при клике на любом месте показываются соответствующие координаты. Ваш код должен при помощи DOM получить четыре пары координат: верхний левый, внешний угол (это просто). нижний правый, внешний угол (тоже просто). верхний левый, внутренний угол (чуть сложнее). нижний правый, внутренний угол (есть несколько способов, выберите один). Координаты, вычисленные вами, должны совпадать с теми, которые возвращаются по клику мыши. P.S. Код должен работать, если у элемента другие размеры или есть рамка, без привязки к конкретным числам. Кликните в любом месте для получения координат в контексте окна.
