### 1.8.1 Напишите функцию showNotification(options), которая создаёт уведомление: div class="notification" с заданным содержимым. Уведомление должно автоматически исчезнуть через 1,5 секунды. showNotification({ top: 10, // 10px от верхней границы окна (по умолчанию 0px) right: 10, // 10px от правого края окна (по умолчанию 0px) html: "Hello!", // HTML-уведомление className: "welcome" // дополнительный класс для div (необязательно) });
